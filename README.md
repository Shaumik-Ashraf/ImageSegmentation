# ImageSegmentation
Image segmentation for glomerulii recognition on kidney tissue

Based on [this Kaggle Competition](https://www.kaggle.com/c/hubmap-kidney-segmentation)

[TransUNet](https://github.com/Beckschen/TransUNet)
